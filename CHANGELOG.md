## [v0.2.0] - 2026-07-19

### Highlights

- AIDLC workflow integration with document generation and phase transitions
- V2MOM cascade management from Organization to Team to Project
- Strategic planning suite with capability stack, roadmap, and maturity model
- Dual methodology selection separating requirements and implementation approaches
- Sample projects with Grafana (comprehensive) and Simple (minimal) examples
- 60+ REST API endpoints for comprehensive feature coverage
